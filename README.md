
# ed25519-python

python binding for [Warchant/ed25519](https://github.com/Warchant/ed25519)

# Need to build

- cmake
- gcc
- g++


# How to install

```
git clone https://github.com/MizukiSonoko/ed25519-python.git
cd ed25519-python
git submodule update --init --recursive
python setup.py develop
```
